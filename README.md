# 📩 Spam Message Detector (Frontend Project)

## 📖 Overview
This is a simple frontend-based project that detects whether a message is **Spam** or **Not Spam** using JavaScript.

It uses a keyword-based approach to simulate how spam detection works in real-world AI systems.

---

## 🎯 Objective
- Detect spam messages using simple logic
- Build an interactive web interface
- Understand basic concept of spam filtering

---

## 🛠️ Technologies Used
- HTML
- CSS
- JavaScript

---

## ⚙️ How It Works
1. User enters a message in the input box
2. JavaScript checks for spam keywords
3. If keywords are found → Spam 🚨
4. Otherwise → Not Spam ✅

---

## 📊 Example

| Input Message | Output |
|--------------|--------|
| You won a free prize | 🚨 Spam |
| Let's meet tomorrow | ✅ Not Spam |
